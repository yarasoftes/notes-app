# Notes App 🚀

Консольное приложение для управления заметками.

## Возможности
- Добавление заметок
- Просмотр всех заметок
- Поиск
- Удаление
- Сохранение в файл

## Запуск
npm install
npm start


# 🚀 Notes App PRO MAX

Консольное приложение для управления заметками на Node.js.

---

## 📦 Установка

Сначала установите зависимости:

```bash
npm install
```

---

## ▶️ Запуск

```bash
npm start
```

---

## 🧠 Используемые библиотеки

* fs (встроенная)
* readline (встроенная)
* chalk

---

## 📖 Команды

```bash
add <текст>       # добавить заметку
list              # показать все заметки
view <номер>      # показать одну заметку
delete <номер>    # удалить заметку
search <текст>    # поиск с подсветкой
stats             # статистика
clear             # очистить экран
help              # список команд
exit              # выход
```

---

## ⚡ Функционал

* 📋 Управление заметками

* ❗ Подтверждение удаления
* 📊 Статистика
* 🎨 Цветной интерфейс
* ⚡ Автодополнение команд (Tab)
